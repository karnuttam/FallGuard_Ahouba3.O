Pattern Recognition

Threshold Monitoring

Continuously compares real-time accelerometer values against pre-defined limits for "Weightlessness" (<0.5g) and "Impact Force" (>2.5g).

False Positive Check

Inactivity Timer

A critical verification step. If the user moves within 10 seconds after impact, the alarm is aborted to prevent false alerts from dropping the phone.

Emergency Protocol

Sequential Action

Triggers a multi-sensory response:
1. Haptic: Vibration for user awareness.
2. Audio: Voice confirmation.
3. Network: GPS fetch & SMS dispatch.
