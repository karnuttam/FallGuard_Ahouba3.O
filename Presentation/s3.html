Project Goals

Key Objectives

Our mission is to create a seamless safety net for the visually impaired through technology.

Sensor-Based Detection

Continuously monitor user movement using smartphone accelerometer and gyroscope to identify fall patterns.

Automated SOS

Trigger immediate emergency alerts via SMS and call to pre-configured contacts without user intervention.

Voice Feedback

Provide clear audio cues and status updates (e.g., "Fall Detected") to reassure visually impaired users.

Haptic Alerts

Use distinct vibration patterns to alert the user of detection events, ensuring awareness even in noisy environments.

Real-Time GPS

Embed precise GPS coordinates in emergency messages to help responders locate the user instantly.

Inclusive Design

Ensure the interface is fully accessible with high contrast, large touch targets, and screen reader compatibility.
